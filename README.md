# Daybreak Asuna Theme

A custom complex dark theme for Visual Studio Code inspired by the warm colors of daybreak.

## Features

- **Warm Sunrise Palette**: Soft oranges, pinks, and golden yellows combined with cool dark backgrounds
- **High Contrast**: Carefully balanced colors for excellent readability
- **Comprehensive Coverage**: Supports syntax highlighting for many popular languages including JavaScript, TypeScript, Python, Go, Rust, C#, Java, and more
- **UI Theming**: Complete workbench theming including editor, sidebar, terminal, and status bar

## Installation

### From Source

1. Clone this repository or download the source
2. Copy the entire folder to your VS Code extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions`
   - **macOS**: `~/.vscode/extensions`
   - **Linux**: `~/.vscode/extensions`
3. Restart VS Code
4. Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
5. Type "Color Theme" and select "Preferences: Color Theme"
6. Choose "Daybreak Asuna" from the list

### From VSIX (Manual Installation)

1. Download the `.vsix` file
2. Open VS Code
3. Go to Extensions view (`Ctrl+Shift+X` or `Cmd+Shift+X`)
4. Click the `...` menu at the top of the Extensions view
5. Select "Install from VSIX..."
6. Select the downloaded `.vsix` file
